Machine learning with Python

This project utilizes the K-Means clustering algorithm from sklearn.cluster to perform image quantization. By defining the number of clusters as 2nbits, where nbits represents the number of bits allocated per pixel, the project effectively compresses images by reducing the color palette. The process is implemented for nbits values of 1, 2, and 6 on a 480×480 RGB image, which is reshaped into a vector containing 230,400 values for clustering. The resulting outputs, stored as centers_ (cluster centers) and labels_ (pixel labels), facilitate a reduction in color complexity while preserving key image details.

GitHub Repository
